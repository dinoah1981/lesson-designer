## Overview
Brief explanation of Robert Marzano's New Taxonomy and why it's used.

## Cognitive Levels (with examples)
1. **Retrieval** - Recognizing, recalling, executing
   - Verbs: list, define, recall, identify, recognize, state, describe
   - Example activities: vocabulary matching, fact recall, basic identification

2. **Comprehension** - Integrating, symbolizing
   - Verbs: summarize, explain, interpret, classify, compare, paraphrase
   - Example activities: concept mapping, summarization, comparison charts

3. **Analysis** - Matching, classifying, analyzing errors, generalizing, specifying
   - Verbs: analyze, differentiate, investigate, categorize, infer, examine
   - Example activities: case analysis, error identification, pattern finding

4. **Knowledge Utilization** - Decision making, problem solving, experimenting, investigating
   - Verbs: design, create, evaluate, hypothesize, apply, construct, produce
   - Example activities: design challenges, investigations, real-world applications

## Lesson Types
Define each lesson type and which Marzano levels are appropriate:
- Introducing new knowledge: Heavy retrieval + comprehension
- Practicing skills: Retrieval + comprehension + some analysis
- Applying knowledge: Analysis + knowledge utilization
- Synthesizing: All levels, emphasis on analysis + utilization
- Novel application: Knowledge utilization dominant

## Cognitive Rigor Requirements
**CRITICAL VALIDATION RULE:**
- Minimum 40% of activities must be higher-order thinking (analysis + knowledge utilization)
- Maximum 30% retrieval-only activities
- Recommended distribution: 15% retrieval, 25% comprehension, 30% analysis, 30% knowledge utilization

## Activity Design Templates
For each Marzano level, provide 3-4 reusable activity templates with:
- Activity name pattern
- Duration range
- Required materials
- Student actions
- Assessment approach

```json
  "materials": ["string"],
  "student_output": "string",
  "assessment_method": "string"
}
```
